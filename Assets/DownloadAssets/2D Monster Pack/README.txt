## 1. Introduction
This asset includes components that require proper setup in Unity.

## 2. Setup Guide
1. Download and import the asset package into your Unity project.
2. Make sure you are using Unity 2022 or newer.
3. Install the Unity 2D package and TextMesh Pro via the Package Manager.

## 3. How to Use
1. Open the ‘2D Monster Pack’ scene located in the ‘SP1/2D Monster Pack/’ folder.
2. Select monster by clicking monster buttons located at the bottom of the screen.
3. Play the monster’s animations by animation buttons located at the top of the screen.
4. Monster prefabs can be found in the 'SP1/2D Monster Pack/MonsterAssets/Monsters/' folder.
